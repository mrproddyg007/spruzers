HTML, CSS and Javascript

HTML

Hyper Text Markup Language(.html)

CSS

CSS - This is used to style the written html and can either be internal or external, its uses the class(.body) and the ID(#body) to implement styling, it is usualy saved using the .css file extension

JS

JS - This is the part of the web page that handles functionalities and dynamism i.e moving of objects, calculations and DOM manipulation and can either be internal or external, its uses the class(.body) and the ID(#body) to handle DOM manipulation, it is usualy saved using the .js file extension.

WEBSITES

Websites are combination of several web pages(html+css+js), Saved in a file and is located(using the url) on a Web Server.
